# Final - Base de Datos
Nombre del Proyecto: Base de Datos Establo
Participantes:
-Joaquin Ojeda Villafuerte
-Luis Sayra Ambilla
-Sareth Marquez Espinoza
Objetivo:
El proyecto busca crear una base de datos para organizar y centralizar la información del establo "El Huarango", mejorando el control y la gestión de sus 230 vacas en producción.

## Tablas (17 entidades)
1. EMPLEADO
2. USUARIO
3. ROL
4. USUARIO_ROL
5. CORRAL
6. VACA
7. HISTORIAL_CORRAL
8. LOTE_PRODUCCION
9. PRODUCCION_LECHE
10. VENTA_LECHE
11. CLIENTE
12. PROVEEDOR
13. INSUMO
14. COMPRA_INSUMO
15. PAGO_EMPLEADO
16. EVENTO_SANITARIO
17. ASISTENCIA
